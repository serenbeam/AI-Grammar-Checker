# O' Grammar Checker

AI-powered grammar checking application built with React Native and TypeScript.

## Overview

O' Grammar Checker is a mobile application designed to help users improve their writing by automatically correcting grammar mistakes and providing explanations for each correction using AI.

The project currently focuses on delivering a simple and clean MVP experience for Android devices while maintaining an architecture that can scale to production-level requirements and future community contributions.

---

## Features

### ✅ Check Screen

* AI-powered grammar checking
* Corrected sentence output
* Detailed explanation of corrections
* Loading and error handling states

### ✅ History Screen

* Displays previously checked grammar results
* History is automatically persisted using local storage
* Data remains available after restarting the application

### ✅ Settings Screen

* Initial settings interface implementation
* Prepared for future functionality and application customization

## Getting Started

### Install dependencies

```bash
npm install
```

### Run Android application

```bash
npm run android
```

---

## Environment Variables

Create a `.env` file in the project root.

Example:

```env
OPENROUTER_API_KEY=your_api_key
OPENROUTER_BASE_URL=https://openrouter.ai/api/v1
```
